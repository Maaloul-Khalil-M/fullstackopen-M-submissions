# Part3 Summary

In this part, we learned about Express.js as a backend and Mongodb as a database
by transforming the "phonebookApp" to a fullstack app, we learned about:

- how the backend fits into the full-stack architecture and communicates with the frontend
- implementing a simple REST API in Node.js by using the Express library
- connecting to a cloud-hosted document database using Mongoose for quering data, schema definition and data validation
- Deployed the REST API to the internet using **Render.com**, making it publicly accessible
  deployment steps documented in ![part3-b.md](./part%203-b.md)
- working with middleware to handle unknown endpoints or errors
- use a linter (ESLint) for **enforcing consistent code style and catching errors early**.
